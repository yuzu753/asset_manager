# Asset Manager Web

ローカルのCSVファイルから月次の金融資産データを読み込み、一覧表示・フィルタ・ソート・カテゴリ別集計ができる Next.js アプリです。  
アプリ本体は `web/` ディレクトリにあります。

## 使い方

1. 依存関係をインストール（ネットワークに接続できる環境で実行してください）。
   ```bash
   cd web
   npm install
   ```
2. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
3. ブラウザで `http://localhost:3000` を開き、CSV ファイルをアップロードしてください。

## CSV のフォーマット

- 1行が1か月分のデータです。
- 先頭行はヘッダー行として扱います（1列目は期間、2列目以降は資産名）。
- 各セルの値は数値（カンマや通貨記号付きでも可）を想定しています。

## 主な機能

- CSV アップロードとクライアントサイドでの解析
- 期間（開始／終了）の指定によるフィルタ
- 指定列での昇順・降順ソート
- 資産ごとのカテゴリ設定とカテゴリ別合計表示
- 資産別合計表示
- カテゴリ別／資産別の横棒グラフ表示（切り替え可能）
- 全資産または選択資産の期間推移を折れ線グラフで可視化

## 補足

- インストール時にネットワークエラーが出た場合は、通信環境を整えてから `npm install` を再実行してください。
